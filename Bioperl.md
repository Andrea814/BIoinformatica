## Instalacion de Bioperl en Ubuntu 18.4
### Primer paso
Colocar en la consola el comando:

`$ sudo apt install bioperl`

### Segundo paso

Se intala el modulo Bio::Seq con el siguiente comando:

`cpan BIo::Seq`
