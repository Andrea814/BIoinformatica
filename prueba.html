Instalación de Python

Paso 1


Se abre una terminal y se ejecuta el siguiente codigo:
$ python --version

Una vez revisando si ya se encuentra preeinstalado el programa solamente se ejecutan los siguientes comandos:

$ sudo apt-get update

$ sudo apt-get install python3.6

Hay que tener en cuenta que al hacer uso del comando Sudo se nos pedira la contraseña es necesario darla para poder continuar con la instalación, posteriormente la terminal nos arrogara un mensaje que diga lo siguiente:

"Python ya se encuentra en su version más reciente (3.6.8-1~18.04.1)", esta version es especifica para mi instalación (imagen 1).

Instalación del PIP

Paso 2

Una ves seguros que tenemos instalados Python se instalará el paquete PIP, iniciando con el comando:

$ sudo apt-get install python3-pip

Esta instalacion es rapida, es necesario tener instalados estos paquetes ya que permitiran instalar los siguientes programas.

Instalación de Git

Paso 3


Este es un repositorio completo de seguimiento de versiones que no dependen del acceso a la red o un servidor.
Para la instalacion se inicia con el comando:
sudo apt install git
